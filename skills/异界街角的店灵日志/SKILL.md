# 异界街角的店灵日志

> 现代灵魂重生为中世纪异世界街角咖啡店的『店灵』，以全知视角观察、间接影响店铺与店员，
> 在经营咖啡店的日常中逐步觉醒，最终面对『以何种形式存在于这个世界』的终极抉择。

触发词：`异界街角的店灵日志，启动！`

## 玩法核心
- 玩家是**店灵**：没有身体、不能说话、不能直接行动，只能对店铺与店员施加**间接影响**。
- 用自然语言描述意图（如『让艾达注意到窗台上的那封信』），调度器会匹配最合适的店灵能力并判定结果（受 SP / 冷却 / 羁绊 / 失败率约束）。
- 经营循环：一日四个时间块（晨备 / 午前营业 / 午后营业 / 打烊后）。说『开店 / 推进 / 打烊』推进时间；不输入则时间不走（帧式）。
- 两种速度：日常模式逐帧；说『快进N天』进入快速模式，脚本自动结算、只看摘要；遇主线事件会强制中断切回。

## 运行架构（脚本驱动数值，LLM 只叙事）
- 所有数值由 `tools/` 下的确定性 Python 脚本计算，LLM 严禁在叙事中直引面板数字（见 AGENTS.md）。
- 每回合引擎 route → `tools/time_block.py`（唯一调度器）：读存档快照 + 玩家意图 → 编排经济/顾客/品质/事件/店灵/进度 → 输出
  `{sim, state_patch, flags_to_set, events, narrative_hint, ...}`。
- 引擎 E1 把 `state_patch` 并入顶层 custom（驱动状态面板）、`flags_to_set` 并入 flags（驱动幕间转换）。

## tools/ 一览
| 文件 | 职责 |
|------|------|
| `time_block.py` | 唯一调度器：意图解析 + 时间块推进 + 子系统编排 |
| `economy.py` | 营业结算、每日结算（薪资/税/维护）、跨夜恢复 |
| `prosperity.py` | 繁荣度增减、口碑、客群分层 |
| `customer_roller.py` | 顾客生成矩阵（种族/年龄/阶层/性格/预算） |
| `quality.py` | 出餐品质与满意度、小费、库存消耗 |
| `menu.py` | 定价公式、菜品自由设计 |
| `spirit.py` | 店灵能力：意图匹配 / SP / 冷却 / 沉睡 / 升级 |
| `event_roller.py` | 事件池 A–F |
| `progression.py` | 主线幕间与 G 类主线就绪判定 |
| `fast_forward.py` | 快速模式多日模拟 + 苏醒摘要 |
| `session_starter.py` / `state_cli.py` | 初始化 / 存档管理 |
| `gamedata.py` / `saveio.py` | 静态数据常量 / 存档读写与面板投影 |

## 阶段
`PROLOGUE → ACT1 → ACT2 → ACT3 → ENDING`，由脚本计算的 flag 驱动转换。
详细设定见同目录 `skill/` 参考文件与 `设计蓝本.txt`（设计文档，非运行文件）。
